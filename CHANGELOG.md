- `dict.update` now returns the dict itself
- implement `--tab`
- implement `--indent N`
- implement `--compact-output`
- implement `--ascii-output` alias
- implement `--raw-output`
- implement `--monochrome-output`
- implement `--arg KEY VALUE`
